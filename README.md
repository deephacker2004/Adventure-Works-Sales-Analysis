# Adventure Works Sales Analysis

## 📌 Project Overview
This project analyzes Adventure Works sales data using SQL, Excel, Power BI, and Tableau. The objective is to transform raw sales data into meaningful business insights through data cleaning, analysis, and interactive dashboards.

---

## 🛠️ Tools & Technologies
- SQL (MySQL)
- Microsoft Excel
- Power BI
- Tableau

---

## 📊 Key KPIs
- Total Sales
- Total Profit
- Profit Margin
- Total Orders
- Average Order Value

---

## 📈 Dashboard Features
- Sales Trend Analysis
- Top Products by Sales
- Top Customers by Sales
- Region-wise Sales Analysis
- Year-wise Sales & Cost Analysis
- Interactive Filters
